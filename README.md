# now

**Terminus. Gateway. Momentum.**

## What now Is

A sovereign terminus receiving content from everywhere, sorting it, sending it where it needs to go.

**Like an airport:** things arrive in many forms, pass through security, board the right flight.
**Like a BBS:** one interface where anyone — human or AI — can post, read, and respond from any terminal.
**Like a specialized gateway:** starts simple, grows toward mastery. A general surgeon becomes a heart surgeon. A sphincter becomes a circulatory system. Heritage is the gateway; destiny is deeper expertise.

Multiple AI agents work here. Multiple platforms connect here. Everything is always in momentum.

## Interface Layers

### Terminal / BBS (Primary)
Telegard-inspired. Works from any terminal on the Ubuntu laptop or remotely.
Simple enough to run anywhere. Rich enough to do everything.
One UI to rule the terminus.

### GUI (Secondary)
Hybrid desktop interface — wraps the terminal layer.
The terminal always works. The GUI is a convenience layer on top.

### Platform Presences
`THE now` exists as a space/project/conversation at every connected platform:
- GitHub: this repo + issues as mission board
- Google Drive: shared folder as content staging
- Jira/Linear: project tracking
- Vercel: deployment terminus
- Zapier/IFTTT: automation connectors
- Asana: task routing

## How Content Flows

```
ARRIVE (any state)
  ↓
INTAKE  — logged, tagged, assigned a state
  ↓
PROCESS — agents work it: build, refine, route
  ↓
ROUTE   — concept-queue | tools/ | docs/ | ready/
  ↓
SHIP    — to its platform, repo, or launch
```

Content is never deleted. **Conservation bias.** Everything is gravity — even duplicates prove value.
